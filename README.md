# TrenchNote (superseded)

**This repository is archived.** Active development is at
**[mds08011/trenchnote](https://github.com/mds08011/trenchnote)**.

This was an early parallel line of TrenchNote. Its unique contributions —
the core/premium boundary ADR, the public API contract, the QR/tag-code
format ADR, and CONTRIBUTING.md — were ported to the main repository in
July 2026 (see ADRs 0010 and 0011 and `docs/API.md` there). Everything
else here is an older version of what the main repo now contains.

Kept read-only for history.
